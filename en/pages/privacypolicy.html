1. General Provisions

This Privacy Policy sets out the rules for the processing and protection of personal data provided by Users in connection with the use of the Fluinty website.

The administrator of personal data contained in the service is Fluinty sp. z o.o. (or the responsible entity). You can contact the Administrator via e-mail: hello@fluinty.com.

3. Purposes and Bases of Processing

Personal data is processed for the purpose of:

  • Providing services electronically (order processing, account management).
  • Communication with the User (contact forms, newsletter - with consent).
  • Analytics and statistics to improve services.
  • Ensuring session and service security.

4. User Rights

Every User has the right to:

  • Access to the content of their data.
  • Rectification, deletion, or limitation of data processing.
  • Object to processing.
  • Data portability.
  • Withdraw consent at any time (without affecting the lawfulness of processing based on consent before its withdrawal).

5. Cookies

The service uses cookies. These are small text files sent by the web server and stored by the browser software. When the browser reconnects to the site, the site recognizes the type of device the user connects from. Parameters allow the information contained in them to be read only by the server that created them.

The User can disable or restore the option of collecting cookies at any time by changing the settings in the web browser.

6. Data Security

The Administrator applies technical and organizational measures ensuring the protection of processed personal data appropriate to the threats and categories of data protected, and in particular protects data against unauthorized disclosure, removal by an unauthorized person, processing in violation of applicable regulations, and alteration, loss, damage, or destruction.

7. Contact

If you have any questions regarding privacy protection, please contact us at the e-mail address: hello@fluinty.com.
